# Typography

Font tokens are build-time SCSS variables. They are resolved at compile time and are not available as CSS custom properties at runtime.

### Font families

| Token | Value |
|---|---|
| `$mg-font-family` | `"Roboto", sans-serif` |
| `$mg-font-family-condensed` | `"Roboto Condensed", sans-serif` |
| `$mg-font-family-headings` | `$mg-font-family-condensed` (overridable via `!default`) |
| `$mg-font-family-icons` | `"fontawesome"` |

### Font sizes
