# Spacing

The Spacing units create visual rhythm and balance.

### When to use:

- To create spacing before, after, or in-between texts.

### States

export const SPACING_TOKENS = [
  '--mg-spacing-0',
  '--mg-spacing-25',
  '--mg-spacing-50',
  '--mg-spacing-75',
  '--mg-spacing-100',
  '--mg-spacing-150',
  '--mg-spacing-175',

  '--mg-spacing-500',

export function SpacingRow({ token }) {
  const value = typeof document !== 'undefined'
    ? getComputedStyle(document.documentElement).getPropertyValue(token).trim()
    : '';
  return (
    <tr>
      <td style={{ padding: '6px 12px', fontFamily: 'monospace', fontSize: '13px' }}>{token}</td>
      <td style={{ padding: '6px 12px', fontSize: '13px' }}>{value || '—'}</td>
      <td style={{ padding: '6px 12px' }}>
        <div style={{ width: value || 0, height: '16px', background: 'var(--mg-color-interactive, #004f91)', borderRadius: '2px', minWidth: parseFloat(value) > 0 ? '2px' : 0 }} />
      </td>
    </tr>
  );
}

### Usage

Standard spacing values are applied across all components. To override spacing in a theme, set `--mg-spacing-*` custom properties on `:root` or inside a `.mg-theme-{name} { }` selector block in your theme file.
