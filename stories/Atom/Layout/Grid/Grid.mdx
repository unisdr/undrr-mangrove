# Grid system

This grid system is based on CSS Grid. The grid comes in two main layouts: auto layout and specified columns layout such as col-2.

<Canvas of={Grid} meta={Grid} />

### Usage

Use the grid to create a layout of your choice based on the design you want to achieve.

- copy the HTML of the variant you want and add the CSS file below to get started.

#### HTML for AUTO LAYOUT VARIANT:

```html
<div class="mg-grid"></div>
```

#### HTML for SPECIFIED COLUMN(S) LAYOUT VARIANT:

```html
<div class="mg-grid mg-grid__col-3"></div>
```

#### CSS:

Add the base layout style from

- https://assets.undrr.org/static/mangrove/1.2.14/assets/css/grid.min.css
