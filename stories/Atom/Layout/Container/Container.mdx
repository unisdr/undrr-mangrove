# Container

Use the `Container` component (`.mg-container`) to establish a basic page width that is responsive.

- `.mg-container--slim` variant that limits width to desktop breakpoint without expanding to wide breakpoint
- `.mg-container--spacer` can be used to apply automatic "smart" spacing between direct child elements
- `.mg-container--padded` adds vertical padding (top and bottom) to the container

## Breakpoints

The component uses the following SASS breakpoints:

- **Mobile**: `$mg-breakpoint-mobile`
- **Tablet**: `$mg-breakpoint-tablet`
- **Desktop**: `$mg-breakpoint-desktop`
- **Desktop wide**: `$mg-breakpoint-desktop`

For more information about breakpoints, see the [Breakpoints story](?path=/docs/design-decisions-breakpoint--docs).

## Example

```jsx
<Container>
  <h1>Responsive Container</h1>
  <p>This container adjusts its width based on the screen size.</p>
</Container>
```
