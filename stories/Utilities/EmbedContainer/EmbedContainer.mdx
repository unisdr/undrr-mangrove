# Embed container

A responsive wrapper for iframes, videos, and other embedded media. Maintains aspect ratio while scaling to fit the container width.

## The problem

Bare `<iframe>` elements have no intrinsic dimensions and don't resize responsively. Previous approaches applied `aspect-ratio: 16/9` to the parent container (e.g. `.mg-highlight-box:has(iframe)`), which forced the *entire* container to 16:9 — breaking layout when the container also held headings, captions, or other content. This was especially broken on Safari/WebKit mobile.

## The solution

Wrap iframes in a `.mg-embed-container` div. The aspect ratio applies only to the wrapper, not the surrounding content.

```html
<div class="mg-embed-container">
  <iframe
    src="https://www.youtube-nocookie.com/embed/VIDEO_ID"
    title="Descriptive title"
    loading="lazy"
    allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share"
    allowfullscreen>
  </iframe>
</div>
```

| Class | Aspect ratio | Use case |
|-------|-------------|----------|
| `mg-embed-container` | 16:9 | Most YouTube/Vimeo videos (default) |
| `mg-embed-container--4x3` | 4:3 | Older video formats, some presentations |
| `mg-embed-container--1x1` | 1:1 | Square video (Instagram-style, YouTube Shorts) |
| `mg-embed-container--21x9` | 21:9 | Ultrawide/cinematic content |

## Inside a highlight box

The recommended pattern for embedding videos in highlight boxes:

<Canvas of={EmbedContainerStories.InsideHighlightBox} />

```html
<aside class="mg-highlight-box mg-highlight-box--secondary mg-highlight-box--centered mb-20">
  <figure>
    <h3>Related video: Video title here</h3>
    <div class="mg-embed-container">
      <iframe
        src="https://www.youtube-nocookie.com/embed/VIDEO_ID"
        loading="lazy"
        title="Related video: Video title here"
        allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share"
        allowfullscreen>
      </iframe>
    </div>
    <figcaption class="mt-10">Caption describing the related video.</figcaption>
  </figure>
</aside>
```

Use `mg-highlight-box--primary` for UNDRR-produced content, `mg-highlight-box--secondary` for external videos.

## Iframe best practices

| Attribute | Value | Why |
|-----------|-------|-----|
| `src` | `youtube-nocookie.com/embed/...` | Privacy-enhanced mode, no tracking cookies until playback |
| `loading` | `lazy` | Performance: only loads when scrolled into view |
| `title` | Descriptive text | WCAG accessibility requirement (SC 4.1.2) |
| `allow` | YouTube permissions | Recommended permissions policy |
| `allowfullscreen` | (boolean) | Enables fullscreen playback |
| `frameborder` | **Do not use** | Deprecated HTML4 attribute; CSS `border: 0` is applied by the embed container |

## Browser support

Uses CSS `aspect-ratio` with a `padding-bottom` percentage fallback for older browsers. Works in all modern browsers including Safari/WebKit mobile.
