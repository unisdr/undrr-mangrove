# Show more

A vanilla JavaScript utility that collapses content behind a "show more" button. This is **not a React component** — it works with plain HTML and is initialized by calling the `mgShowMore()` function. The React wrapper in Storybook is only used here for demonstration purposes.

## Default

The button is placed directly after the collapsible content.

<Canvas of={ShowMoreStories.DefaultShowMore} />

## Detached button

The toggle button does not need to be inside or adjacent to the content container. Use `data-mg-show-more-target` to point it at any element on the page by CSS selector.

<Canvas of={ShowMoreStories.DetachedButton} />

## How it works

The `mgShowMore()` function finds all elements with `data-mg-show-more` and wires up click handlers. On click, the targeted container's `mg-show-more--collapsed` class is toggled, which controls `max-height`, `overflow`, and a CSS `mask-image` fade.

### HTML structure

<Source language="html" code={`
<!-- Content to collapse -->
<div class="my-content">
  <p>Long content goes here...</p>
</div>

<!-- Toggle button (can be anywhere on the page) -->
<a href="#"
   class="mg-button mg-button-primary mg-show-more--button"
   data-mg-show-more="true"
   data-mg-show-more-target=".my-content"
   data-mg-show-more-label-open="Show less"
   data-mg-show-more-label-collapsed="Show more">
  Show more
</a>
`} />

### Data attributes

| Attribute | Required | Description |
|---|---|---|
| `data-mg-show-more` | Yes | Set to `"true"` on the toggle button to register it |
| `data-mg-show-more-target` | No | CSS selector for the content container. Defaults to `".mg-show-more--container"` |
| `data-mg-show-more-label-collapsed` | No | Button text when content is collapsed. Defaults to "Show more" |
| `data-mg-show-more-label-open` | No | Button text when content is expanded. Defaults to "Show less" |

### JavaScript initialization

<Source language="html" code={`
<!-- Load the script -->
<script src="https://assets.undrr.org/static/mangrove/1.3.3/js/show-more.js"></script>

<!-- The script auto-initializes on DOMContentLoaded. -->
<!-- For dynamically added content, call mgShowMore() manually: -->
<script>
  mgShowMore();
</script>
`} />

### Customizing the collapse height

The collapsed height is controlled by the `--mg-show-more-height` CSS custom property (default: `200px`):

<Source language="css" code={`
/* Override globally */
:root {
  --mg-show-more-height: 150px;
}

/* Or per instance */
.my-content {
  --mg-show-more-height: 300px;
}
`} />
