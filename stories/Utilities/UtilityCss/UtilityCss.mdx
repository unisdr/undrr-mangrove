# Utility classes

Mangrove provides a growing collection of utility classes for common styling and layout needs.

## Sass

Sass/CSS is located at `stories/assets/scss/_utility.scss`

## Screen reader only (.mg-u-sr-only)

This utility class hides content visually while keeping it accessible to screen readers.

```html
<span class="mg-u-sr-only">Screen reader only text</span>
```

## Responsive display utilities

Control element visibility based on screen size.

### Show on large screens only (.mg-u-responsive--show-large)

Hides content on small screens, shows on medium screens and up.

```html
<div class="mg-u-responsive--show-large">This content only appears on larger screens</div>
```

### Show on small screens only (.mg-u-responsive--show-small)

Shows content on small screens, hides on medium screens and up.

```html
<div class="mg-u-responsive--show-small">This content only appears on smaller screens</div>
```

## Layout utilities

### Expand to size (.mg-u-expand-to-size)

Creates responsive container widths with margin adjustments.

```html
<div class="mg-u-expand-to-size">Responsive container content</div>
```

### Overflow hidden (.mg-u-overflow-hidden)

Hides content that overflows the container boundaries.

```html
<div class="mg-u-overflow-hidden">Content that won't show scrollbars</div>
```

## Text wrapping utilities

### Balanced text wrap (.mg-u-text-wrap-balanced)

Creates balanced line breaks for more aesthetically pleasing text layout.

```html
<div class="mg-u-text-wrap-balanced">Your text content here</div>
```

### Pretty text wrap (.mg-u-text-wrap-pretty)

Optimizes text wrapping for improved readability.

```html
<div class="mg-u-text-wrap-pretty">Your text content here</div>
```

### Comma separator (.mg-u-comma-between)

Automatically adds commas between child elements, useful for lists.

```html
<div class="mg-u-comma-between">
  <span>Item 1</span>
  <span>Item 2</span>
  <span>Item 3</span>
</div>
```
