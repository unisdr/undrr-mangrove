# Utility classes

Mangrove provides a growing collection of utility classes for common styling and layout needs.

## Sass

Sass/CSS is located at `stories/assets/scss/_utility.scss`

## Screen reader only (.mg-u-sr-only)

This utility class hides content visually while keeping it accessible to screen readers.

```html
<span class="mg-u-sr-only">Screen reader only text</span>
```

## Responsive display utilities

Control element visibility based on screen size.

### Show on large screens only (.mg-u-responsive--show-large)

Hides content on small screens, shows on medium screens and up.

```html
<div class="mg-u-responsive--show-large">This content only appears on larger screens</div>
```

### Show on small screens only (.mg-u-responsive--show-small)

Shows content on small screens, hides on medium screens and up.

```html
<div class="mg-u-responsive--show-small">This content only appears on smaller screens</div>
```

## Layout utilities

### Expand to size (.mg-u-expand-to-size)

Creates responsive container widths with margin adjustments.

```html
<div class="mg-u-expand-to-size">Responsive container content</div>
```

### Overflow hidden (.mg-u-overflow-hidden)

Hides content that overflows the container boundaries.

```html
<div class="mg-u-overflow-hidden">Content that won't show scrollbars</div>
```

## Text wrapping utilities

### Balanced text wrap (.mg-u-text-wrap-balanced)

Creates balanced line breaks for more aesthetically pleasing text layout.

```html
<div class="mg-u-text-wrap-balanced">Your text content here</div>
```

### Pretty text wrap (.mg-u-text-wrap-pretty)

Optimizes text wrapping for improved readability.

```html
<div class="mg-u-text-wrap-pretty">Your text content here</div>
```

### Comma separator (.mg-u-comma-between)

Automatically adds commas between child elements, useful for lists.

```html
<div class="mg-u-comma-between">
  <span>Item 1</span>
  <span>Item 2</span>
  <span>Item 3</span>
</div>
```

## Flex utilities

Basic flexbox layout utilities for composing inline layouts.

<Canvas of={UtilityCssStories.FlexUtilities} />

| Class | CSS | Description |
|-------|-----|-------------|
| `.mg-u-flex` | `display: flex` | Enables flexbox on the element |
| `.mg-u-flex-wrap` | `flex-wrap: wrap` | Allows flex items to wrap |
| `.mg-u-align-items-center` | `align-items: center` | Vertically centers flex items |
| `.mg-u-gap-100` | `gap: 1rem` | Adds 1rem gap between flex items |

```html
<div class="mg-u-flex mg-u-align-items-center mg-u-gap-100">
  <span>Item 1</span>
  <span>Item 2</span>
</div>
```

## Alignment push utilities

RTL-safe utilities to push an element toward one side of its container using CSS logical properties (`margin-inline-start`, `margin-inline-end`). These automatically flip in RTL contexts. Pair with `.mg-u-flex` for inline layouts.

<Canvas of={UtilityCssStories.PushUtilities} />

| Class | CSS | Effect |
|-------|-----|--------|
| `.mg-u-push-end` | `margin-inline-start: auto` | Pushes element to the inline end (right in LTR, left in RTL) |
| `.mg-u-push-start` | `margin-inline-end: auto` | Pushes element to the inline start (left in LTR, right in RTL) |
| `.mg-u-push-center` | `margin-inline: auto` | Centers element horizontally |

### Examples

Push a button to the end of a flex row:

```html
<div class="mg-u-flex mg-u-align-items-center">
  <h2>How it works</h2>
  <a href="#" class="mg-button mg-button-primary mg-u-push-end">View all</a>
</div>
```

Center a block element:

```html
<img src="logo.svg" class="mg-u-push-center" alt="Logo" />
```

## Font size utilities (.mg-u-font-size-*)

Responsive font-size classes based on Mangrove design tokens. Each class sets a mobile-first base size and scales up at the medium breakpoint (48em / 768px).

| Class | Mobile | Desktop (>=48em) | Line height |
|-------|--------|-------------------|-------------|
| `.mg-u-font-size-150` | 1.125rem | 1.125rem | 1.5em |
| `.mg-u-font-size-200` | 1.25rem | 1.25rem | 1.5em |
| `.mg-u-font-size-250` | 1.4rem | 1.4rem | 1.5em |
| `.mg-u-font-size-300` | 1.6rem | 1.6rem | 1.5em |
| `.mg-u-font-size-400` | 1.6rem | 1.8rem | 1.5em |
| `.mg-u-font-size-500` | 1.8rem | 2.3rem | 1.25em |
| `.mg-u-font-size-600` | 2.3rem | 3.2rem | 1.25em |
| `.mg-u-font-size-800` | 3.2rem | 3.6rem | 1.25em |
| `.mg-u-font-size-900` | 3.6rem | 4rem | 1.25em |

Language-specific line-height overrides are included for Burmese (`:lang(my)` — 1.7) and Arabic (`:lang(ar)` — 1.4) on larger sizes.

```html
<p class="mg-u-font-size-600">Large responsive text</p>
<h2 class="mg-u-font-size-900">Override heading size</h2>
```
