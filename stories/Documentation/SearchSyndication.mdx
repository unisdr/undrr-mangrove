# Search syndication

Mangrove includes a full-featured search interface for querying content across UNDRR domains (undrr.org, preventionweb.net, and related sites). The `SyndicationSearchWidget` connects to UNDRR's Elasticsearch backend and provides faceted filtering, card and list display modes, and URL hash sync.

A search widget can be embedded on any page with a container element:

```html
<div data-mg-search-widget
  data-search-endpoint="https://www.undrr.org/search-endpoint"
  data-results-per-page="50"
  data-show-facets="true"
  data-default-filters='[{"key":"_language","value":"en"}]'>
</div>
```

The search scope can be constrained with `customFilters` (hidden Elasticsearch queries) and extended with `customFacets` (user-selectable dropdowns). Content types, date ranges, and taxonomies can all be filtered.

> **Platform dependency:** The search widget requires access to the UNDRR Elasticsearch backend via `/search-endpoint`. It is not a general-purpose search component.

See the <LinkTo kind="components-syndicationsearchwidget" story="docs">SyndicationSearchWidget component</LinkTo> for full documentation, props, query syntax, custom facets, and integration examples.
