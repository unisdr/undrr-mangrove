# Spacing utilities

Spacing utilities control margin and padding using Mangrove's spacing tokens. Mangrove uses a 10px base font size, so `1rem = 10px`.

## Naming convention

```
.mg-u-{margin|padding}-{direction}-{size}
.mg-u-{margin|padding}-{direction}-{size}--{breakpoint}
```

Where:
- **direction**: `top`, `right`, `bottom`, `left`, `x` (left + right), `y` (top + bottom), or omit for all sides
- **size**: Token suffix (0, 25, 50, 75, 100, 150, 200, 250, 300, 400, 500, 600, 800)
- **breakpoint**: `sm`, `md`, `lg`, `xl` (optional, for responsive)

## Spacing scale

| Token | Class suffix | Value | Pixels |
|-------|--------------|-------|--------|
| `$mg-spacing-0` | `-0` | 0 | 0px |
| `$mg-spacing-25` | `-25` | 0.25rem | 2.5px |
| `$mg-spacing-50` | `-50` | 0.5rem | 5px |
| `$mg-spacing-75` | `-75` | 0.75rem | 7.5px |
| `$mg-spacing-100` | `-100` | 1rem | 10px |
| `$mg-spacing-150` | `-150` | 1.5rem | 15px |
| `$mg-spacing-175` | `-175` | 1.8rem | 18px |
| `$mg-spacing-200` | `-200` | 2rem | 20px |
| `$mg-spacing-250` | `-250` | 2.4rem | 24px |
| `$mg-spacing-300` | `-300` | 3rem | 30px |
| `$mg-spacing-400` | `-400` | 5rem | 50px |
| `$mg-spacing-500` | `-500` | 6rem | 60px |
| `$mg-spacing-600` | `-600` | 8rem | 80px |
| `$mg-spacing-800` | `-800` | 10rem | 100px |

## Margin utilities

### All sides

```html
<div class="mg-u-margin-0">No margin</div>
<div class="mg-u-margin-100">10px margin all sides</div>
<div class="mg-u-margin-200">20px margin all sides</div>
<div class="mg-u-margin-300">30px margin all sides</div>
```

### Individual sides

```html
<div class="mg-u-margin-top-100">margin-top: 10px</div>
<div class="mg-u-margin-right-150">margin-right: 15px</div>
<div class="mg-u-margin-bottom-200">margin-bottom: 20px</div>
<div class="mg-u-margin-left-250">margin-left: 24px</div>
```

### Horizontal and vertical

```html
<div class="mg-u-margin-x-200">margin-left: 20px; margin-right: 20px</div>
<div class="mg-u-margin-y-300">margin-top: 30px; margin-bottom: 30px</div>
```

### Auto margins

```html
<div class="mg-u-margin-auto">margin: auto</div>
<div class="mg-u-margin-x-auto">Center horizontally</div>
<div class="mg-u-margin-y-auto">Center vertically</div>
<div class="mg-u-margin-top-auto">Push to bottom (in flex)</div>
<div class="mg-u-margin-left-auto">Push to right (in flex)</div>
```

### Negative margins

Negative margins are available for layout adjustments:

```html
<div class="mg-u-margin-top-negative-100">margin-top: -10px</div>
<div class="mg-u-margin-top-negative-200">margin-top: -20px</div>
<div class="mg-u-margin-bottom-negative-100">margin-bottom: -10px</div>
<div class="mg-u-margin-left-negative-200">margin-left: -20px</div>
```

## Responsive spacing

Add breakpoint suffixes for responsive spacing:

| Suffix | Breakpoint | Description |
|--------|------------|-------------|
| `--sm` | ≤480px | Mobile only |
| `--md` | ≥900px | Tablet and up |
| `--lg` | ≥1164px | Desktop and up |
| `--xl` | ≥1440px | Wide desktop |

### Examples

```html
<!-- Smaller margin on mobile, larger on desktop -->
<div class="mg-u-margin-top-100 mg-u-margin-top-300--md mg-u-margin-top-400--lg">
  Responsive margin
</div>

<!-- More padding on larger screens -->
<div class="mg-u-padding-y-100 mg-u-padding-y-200--md mg-u-padding-y-300--lg">
  Responsive padding
</div>

<!-- Reduce padding on mobile -->
<div class="mg-u-padding-x-50--sm mg-u-padding-x-200--md">
  Less padding on mobile
</div>
```

## Complete class reference

### Margin classes

| Class | CSS output |
|-------|------------|
| `.mg-u-margin-{size}` | `margin: {value}` |
| `.mg-u-margin-top-{size}` | `margin-top: {value}` |
| `.mg-u-margin-right-{size}` | `margin-right: {value}` |
| `.mg-u-margin-bottom-{size}` | `margin-bottom: {value}` |
| `.mg-u-margin-left-{size}` | `margin-left: {value}` |
| `.mg-u-margin-x-{size}` | `margin-left: {value}; margin-right: {value}` |
| `.mg-u-margin-y-{size}` | `margin-top: {value}; margin-bottom: {value}` |
| `.mg-u-margin-auto` | `margin: auto` |
| `.mg-u-margin-x-auto` | `margin-left: auto; margin-right: auto` |
| `.mg-u-margin-y-auto` | `margin-top: auto; margin-bottom: auto` |
| `.mg-u-margin-{dir}-negative-{size}` | `margin-{dir}: -{value}` |

### Padding classes

| Class | CSS output |
|-------|------------|
| `.mg-u-padding-{size}` | `padding: {value}` |
| `.mg-u-padding-top-{size}` | `padding-top: {value}` |
| `.mg-u-padding-right-{size}` | `padding-right: {value}` |
| `.mg-u-padding-bottom-{size}` | `padding-bottom: {value}` |
| `.mg-u-padding-left-{size}` | `padding-left: {value}` |
| `.mg-u-padding-x-{size}` | `padding-left: {value}; padding-right: {value}` |
| `.mg-u-padding-y-{size}` | `padding-top: {value}; padding-bottom: {value}` |
