# Flexbox utilities

Control flex container and item behavior with flexbox utilities.

## Flex direction

Control the direction of flex items:

```html
<div class="mg-u-display--flex mg-u-flex-direction--row">Row (default)</div>
<div class="mg-u-display--flex mg-u-flex-direction--column">Column</div>
```

### Responsive flex direction

```html
<!-- Column on mobile, row on tablet+ -->
<div
  class="mg-u-display--flex mg-u-flex-direction--column mg-u-flex-direction--row--md"
>
  <div>Item 1</div>
  <div>Item 2</div>
</div>
```

## Flex wrap

Control whether flex items wrap:

```html
<div class="mg-u-display--flex mg-u-flex-wrap--wrap">Wrap items</div>
<div class="mg-u-display--flex mg-u-flex-wrap--nowrap">No wrap (default)</div>
```

## Justify content

Control alignment along the main axis:

```html
<div class="mg-u-display--flex mg-u-justify-content--start">
  Start (default)
</div>
<div class="mg-u-display--flex mg-u-justify-content--end">End</div>
<div class="mg-u-display--flex mg-u-justify-content--center">Center</div>
<div class="mg-u-display--flex mg-u-justify-content--between">
  Space between
</div>
```

## Align items

Control alignment along the cross axis:

```html
<div class="mg-u-display--flex mg-u-align-items--start">Start</div>
<div class="mg-u-display--flex mg-u-align-items--end">End</div>
<div class="mg-u-display--flex mg-u-align-items--center">Center</div>
<div class="mg-u-display--flex mg-u-align-items--stretch">
  Stretch (default)
</div>
```

## Gap

Control spacing between flex items:

```html
<div class="mg-u-display--flex mg-u-gap-0">No gap</div>
<div class="mg-u-display--flex mg-u-gap-50">5px gap</div>
<div class="mg-u-display--flex mg-u-gap-100">10px gap</div>
<div class="mg-u-display--flex mg-u-gap-200">20px gap</div>
<div class="mg-u-display--flex mg-u-gap-300">30px gap</div>
<div class="mg-u-display--flex mg-u-gap-400">50px gap</div>
```

## Flex item utilities

```html
<div class="mg-u-flex--1">Fill available space equally</div>
<div class="mg-u-flex--none">Don't grow or shrink</div>
<div class="mg-u-flex-shrink--0">Prevent shrinking</div>
```

## Common patterns

### Centered content

```html
<div
  class="mg-u-display--flex mg-u-justify-content--center mg-u-align-items--center"
>
  Centered both axes
</div>
```

### Space between with wrap

```html
<div
  class="mg-u-display--flex mg-u-justify-content--between mg-u-flex-wrap--wrap mg-u-gap-200"
>
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
</div>
```

### Responsive layout

```html
<!-- Stack on mobile, side-by-side on tablet+ -->
<div
  class="mg-u-display--flex mg-u-flex-direction--column mg-u-flex-direction--row--md mg-u-gap-200"
>
  <div class="mg-u-flex--1">Column 1</div>
  <div class="mg-u-flex--1">Column 2</div>
</div>
```

### Navigation bar

```html
<nav
  class="mg-u-display--flex mg-u-justify-content--between mg-u-align-items--center"
>
  <div>Logo</div>
  <div class="mg-u-display--flex mg-u-gap-200">
    <a href="#">Link 1</a>
    <a href="#">Link 2</a>
    <a href="#">Link 3</a>
  </div>
</nav>
```

## Complete class reference

### Container classes

| Class                              | CSS output                         |
| ---------------------------------- | ---------------------------------- |
| `.mg-u-flex-direction--row`        | `flex-direction: row`              |
| `.mg-u-flex-direction--column`     | `flex-direction: column`           |
| `.mg-u-flex-direction--row--md`    | `flex-direction: row` (tablet+)    |
| `.mg-u-flex-direction--column--md` | `flex-direction: column` (tablet+) |
| `.mg-u-flex-wrap--wrap`            | `flex-wrap: wrap`                  |
| `.mg-u-flex-wrap--nowrap`          | `flex-wrap: nowrap`                |
| `.mg-u-justify-content--start`     | `justify-content: flex-start`      |
| `.mg-u-justify-content--end`       | `justify-content: flex-end`        |
| `.mg-u-justify-content--center`    | `justify-content: center`          |
| `.mg-u-justify-content--between`   | `justify-content: space-between`   |
| `.mg-u-align-items--start`         | `align-items: flex-start`          |
| `.mg-u-align-items--end`           | `align-items: flex-end`            |
| `.mg-u-align-items--center`        | `align-items: center`              |
| `.mg-u-align-items--stretch`       | `align-items: stretch`             |
