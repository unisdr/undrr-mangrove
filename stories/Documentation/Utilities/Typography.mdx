# Typography utilities

Control font size, weight, alignment, and text styling with typography utilities.

## Font size

Font sizes use Mangrove's typography tokens. With a 10px base, sizes scale proportionally:

| Class                  | Token                | Value    | Approx. pixels |
| ---------------------- | -------------------- | -------- | -------------- |
| `.mg-u-font-size-100`  | `$mg-font-size-100`  | 1rem     | 10px           |
| `.mg-u-font-size-150`  | `$mg-font-size-150`  | 1.125rem | 11px           |
| `.mg-u-font-size-200`  | `$mg-font-size-200`  | 1.25rem  | 12.5px         |
| `.mg-u-font-size-250`  | `$mg-font-size-250`  | 1.4rem   | 14px           |
| `.mg-u-font-size-300`  | `$mg-font-size-300`  | 1.6rem   | 16px (body)    |
| `.mg-u-font-size-400`  | `$mg-font-size-400`  | 1.8rem   | 18px           |
| `.mg-u-font-size-500`  | `$mg-font-size-500`  | 2.3rem   | 23px           |
| `.mg-u-font-size-600`  | `$mg-font-size-600`  | 3.2rem   | 32px           |
| `.mg-u-font-size-800`  | `$mg-font-size-800`  | 3.6rem   | 36px           |
| `.mg-u-font-size-900`  | `$mg-font-size-900`  | 4rem     | 40px           |
| `.mg-u-font-size-1000` | `$mg-font-size-1000` | 4.8rem   | 48px           |
| `.mg-u-font-size-1100` | `$mg-font-size-1100` | 6.4rem   | 64px           |

```html
<p class="mg-u-font-size-300">Body text (16px)</p>
<h2 class="mg-u-font-size-600">Heading (32px)</h2>
<span class="mg-u-font-size-200">Small text (12.5px)</span>
```

### Responsive font sizes

```html
<!-- Larger font on desktop -->
<h1 class="mg-u-font-size-500 mg-u-font-size-600--lg">Responsive heading</h1>
```

## Font weight

```html
<p class="mg-u-font-weight--normal">Normal (400)</p>
<p class="mg-u-font-weight--semibold">Semibold (600)</p>
<p class="mg-u-font-weight--bold">Bold (700)</p>
```

## Font family

```html
<p class="mg-u-font-family--base">Roboto (default)</p>
<p class="mg-u-font-family--condensed">Roboto Condensed</p>
```

## Font style

```html
<p class="mg-u-font-style--normal">Normal text</p>
<p class="mg-u-font-style--italic">Italic text</p>
```

## Text alignment

```html
<p class="mg-u-text-align--left">Left aligned</p>
<p class="mg-u-text-align--center">Center aligned</p>
<p class="mg-u-text-align--right">Right aligned</p>
```

### Responsive alignment

```html
<!-- Center on mobile, left on tablet+ -->
<p class="mg-u-text-align--center--sm mg-u-text-align--left--md">
  Responsive alignment
</p>
```

## Line height

```html
<p class="mg-u-line-height--none">Line height: 1</p>
<p class="mg-u-line-height--tight">Line height: 1.25</p>
<p class="mg-u-line-height--normal">Line height: 1.5</p>
<p class="mg-u-line-height--relaxed">Line height: 1.75</p>
<p class="mg-u-line-height--loose">Line height: 2</p>
```

## Text transform

```html
<p class="mg-u-text-transform--uppercase">UPPERCASE TEXT</p>
<p class="mg-u-text-transform--lowercase">lowercase text</p>
<p class="mg-u-text-transform--capitalize">Capitalize Each Word</p>
<p class="mg-u-text-transform--none">Normal text</p>
```

## White space

```html
<p class="mg-u-white-space--normal">Normal white space handling</p>
<p class="mg-u-white-space--nowrap">No wrapping allowed</p>
<p class="mg-u-white-space--pre">Preserves whitespace</p>
<p class="mg-u-white-space--pre-line">Preserves newlines</p>
<p class="mg-u-white-space--pre-wrap">Preserves whitespace, wraps</p>
<p class="mg-u-white-space--break-spaces">Preserves spaces, breaks anywhere</p>
```

## Text truncation

### Single line truncation

```html
<p class="mg-u-truncate" style="width: 200px;">
  This long text will be truncated with an ellipsis...
</p>
```

### Text overflow

```html
<p class="mg-u-text-overflow--ellipsis">Ellipsis on overflow</p>
<p class="mg-u-text-overflow--clip">Clip on overflow</p>
```

## Word break and overflow wrap

```html
<p class="mg-u-word-break--normal">Normal word breaking</p>
<p class="mg-u-word-break--break-all">Break anywhere</p>
<p class="mg-u-word-break--keep-all">Don't break CJK text</p>

<p class="mg-u-overflow-wrap--normal">Normal overflow wrap</p>
<p class="mg-u-overflow-wrap--break-word">Break long words</p>
<p class="mg-u-overflow-wrap--anywhere">Break anywhere if needed</p>
```

<p class="mg-u-letter-spacing--tight">Tight (-0.025em)</p>

## Common patterns

### Article typography

```html
<article>
  <h1 class="mg-u-font-size-800 mg-u-font-weight--bold mg-u-line-height--tight">
    Article title
  </h1>
  <p class="mg-u-font-size-400 mg-u-color--neutral-600 mg-u-margin-bottom-300">
    Article subtitle or description
  </p>
  <div class="mg-u-font-size-300 mg-u-line-height--relaxed">
    <p>Article content...</p>
  </div>
</article>
```

### Responsive hero text

```html
<h1 class="mg-u-font-size-500 mg-u-font-size-600--lg mg-u-text-align--center">
  Hero headline
</h1>
```
