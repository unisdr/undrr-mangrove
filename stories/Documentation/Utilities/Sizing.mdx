# Sizing utilities

Control element width, height, and aspect ratio with sizing utilities.

## Width

### Keyword widths

```html
<div class="mg-u-width--auto">Width: auto</div>
<div class="mg-u-width--full">Width: 100%</div>
<div class="mg-u-width--screen">Width: 100vw</div>
<div class="mg-u-width--min">Width: min-content</div>
<div class="mg-u-width--max">Width: max-content</div>
<div class="mg-u-width--fit">Width: fit-content</div>
```

### Responsive widths

```html
<!-- Full width on mobile, auto on tablet+ -->
<div class="mg-u-width--full--sm mg-u-width--auto--md">Responsive width</div>
```

## Min/max width

### Min width

```html
<div class="mg-u-min-width--0">min-width: 0</div>
<div class="mg-u-min-width--full">min-width: 100%</div>

<div class="mg-u-min-width--max">min-width: max-content</div>
```

## Size (width and height together)

Set both dimensions at once:

```html
<div class="mg-u-size--full">100% x 100%</div>
<div class="mg-u-size--screen">100vw x 100vh</div>
<div class="mg-u-size--min">min-content x min-content</div>
<div class="mg-u-size--max">max-content x max-content</div>
<div class="mg-u-size--fit">fit-content x fit-content</div>
```

## Aspect ratio

Maintain consistent proportions:

```html
<div class="mg-u-aspect--auto">aspect-ratio: auto</div>
<div class="mg-u-aspect--square">aspect-ratio: 1/1</div>
<div class="mg-u-aspect--video">aspect-ratio: 16/9</div>
<div class="mg-u-aspect--4-3">aspect-ratio: 4/3</div>
<div class="mg-u-aspect--3-2">aspect-ratio: 3/2</div>
<div class="mg-u-aspect--2-1">aspect-ratio: 2/1</div>
<div class="mg-u-aspect--21-9">aspect-ratio: 21/9 (ultrawide)</div>
<div class="mg-u-aspect--portrait">aspect-ratio: 3/4</div>
```

## Common patterns

### Full-screen hero

```html
<section
  class="mg-u-min-height--screen mg-u-display--flex mg-u-align-items--center"
>
  <div class="mg-u-max-width--3xl mg-u-margin-x-auto">Hero content</div>
</section>
```

### Constrained article

```html
<article class="mg-u-max-width--prose mg-u-margin-x-auto">
  <p>Article text with optimal reading width...</p>
</article>
```

### Video container

```html
<div class="mg-u-aspect--video mg-u-width--full">
  <iframe class="mg-u-size--full" src="video.mp4"></iframe>
</div>
```

### Card image

```html
<div class="mg-u-aspect--4-3 mg-u-overflow--hidden">
  <img
    class="mg-u-width--full mg-u-height--full mg-u-object-fit--cover"
    src="card.jpg"
    alt=""
  />
</div>
```
