# CTA

CTA block defines a section of information data with some items - title, button/cta and description. `H2`,  `P` and `Button` atoms tags are used for creating this section.

### Usage

* Take HTML from the HTML tab in canvas
* Include CSS and JS from the 'CSS and JS References' section

### CSS and JS References

#### CSS:

Add the base style only, located at **dist/css/base.min.css** along with following:

* https://unisdr.github.io/undrr-mangrove/css/components/textcta.min.css

#### JS:

* https://unisdr.github.io/undrr-mangrove/js/viewport.min.js (optional for viewport animation)
