# CTA

A full-width call-to-action banner with heading, rich text body, and action buttons. Uses the same color variant system as Hero (primary, secondary, tertiary, quaternary) and supports custom background colors.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `headline` | String | - | Banner heading text |
| `text` | String | - | Body text (HTML supported via `dangerouslySetInnerHTML`) |
| `buttons` | Array | `[]` | Array of button objects: `{ label, url, type }` |
| `variant` | String | `'primary'` | Color variant: `'primary'`, `'secondary'`, `'tertiary'`, `'quaternary'` |
| `backgroundColor` | String | - | Custom CSS background color (overrides variant) |
| `image` | String | - | Image URL displayed alongside text (triggers side-by-side layout) |
| `imageAlt` | String | `''` | Alt text for the image |
| `centered` | Boolean | `true` | Center-align content (auto-disabled when `image` is set) |
| `className` | String | - | Additional CSS classes |

### Button object

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `label` | String | Required | Button text |
| `url` | String | `'#'` | Button link URL |
| `type` | String | `'Primary'` | Button style: `'Primary'` or `'Secondary'` |

## Accessibility

- Uses semantic `<section>` element
- Heading hierarchy preserved (renders `<h2>`)
- Button text provides clear action labels
- Color contrast: white text on theme background colors meets WCAG AA
- Focus states inherited from `.mg-button` base styles
