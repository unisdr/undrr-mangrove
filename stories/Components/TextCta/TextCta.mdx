# CTA

A full-width call-to-action banner with heading, rich text body, and action buttons. Uses the same color variant system as Hero (primary, secondary, tertiary, quaternary) and supports custom background colors.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `headline` | String | - | Banner heading text |
| `headlineSize` | String | `'600'` | Font size token applied as `mg-u-font-size-{value}` (e.g. `'400'`, `'600'`, `'800'`) |
| `headlineLevel` | Number | `2` | Semantic heading level (2–6). Controls the HTML element independently of visual size |
| `text` | String | - | Body text (HTML supported via `dangerouslySetInnerHTML`) |
| `buttons` | Array | `[]` | Array of button objects: `{ label, url, type }` |
| `variant` | String | `'primary'` | Color variant: `'primary'`, `'secondary'`, `'tertiary'`, `'quaternary'` |
| `backgroundColor` | String | - | Custom CSS background color (overrides variant) |
| `padding` | String | - | Custom CSS padding (e.g. `'8rem 0'`), overrides theme token |
| `image` | String | - | Image URL displayed alongside text (triggers side-by-side layout) |
| `imageAlt` | String | `''` | Alt text for the image |
| `centered` | Boolean | `true` | Center-align content (auto-disabled when `image` is set) |
| `className` | String | - | Additional CSS classes |

### Button object

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `label` | String | Required | Button text |
| `url` | String | `'#'` | Button link URL |
| `type` | String | `'Primary'` | Button style: `'Primary'` or `'Secondary'` |

## Server-rendered / CDN usage

TextCta supports [layered hydration](https://github.com/unisdr/undrr-mangrove/blob/main/docs/HYDRATION.md). Use data attributes on a container element:

```html
<div data-mg-text-cta
  data-headline="Join the global platform"
  data-text="<p>Register now for the Global Platform.</p>"
  data-buttons='[{"label":"Register","url":"/register"},{"label":"Learn more","url":"/about","type":"Secondary"}]'
  data-variant="primary">
</div>
```

| Attribute | Default | Description |
|-----------|---------|-------------|
| `data-headline` | `''` | Banner heading text |
| `data-headline-size` | `'600'` | Font size token (e.g. `'400'`, `'600'`, `'800'`) |
| `data-headline-level` | `2` | Semantic heading level (2–6) |
| `data-text` | `''` | Body HTML (sanitized via DOMPurify) |
| `data-buttons` | `[]` | JSON array of button objects: `{ label, url, type }` |
| `data-variant` | `'primary'` | `'primary'`, `'secondary'`, `'tertiary'`, `'quaternary'` |
| `data-background-color` | - | Custom CSS background color |
| `data-padding` | - | Custom CSS padding |
| `data-image` | - | Image URL (triggers side-by-side layout) |
| `data-image-alt` | `''` | Alt text for the image |
| `data-centered` | `'true'` | `'true'` or `'false'` |

```js
import createHydrator from "https://assets.undrr.org/static/mangrove/1.5.0/components/hydrate.js";
import TextCta, { fromElement } from "https://assets.undrr.org/static/mangrove/1.5.0/components/TextCta.js";
createHydrator({ selector: "[data-mg-text-cta]", component: TextCta, fromElement });
```

See the [hydration guide](https://github.com/unisdr/undrr-mangrove/blob/main/docs/HYDRATION.md) for the full API and integration details.

## Accessibility

- Uses semantic `<section>` element with `aria-labelledby` pointing to the headline (falls back to `aria-label="Call to action"` when no headline is provided)
- Headline renders as a proper heading element (`<h2>` by default). Use `headlineLevel` to match the document outline
- Button text provides clear action labels
- Color contrast: white text on theme background colors meets WCAG AA
- Focus states inherited from `.mg-button` base styles
