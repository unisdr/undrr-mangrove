# Page template example

This is the `PageTemplateExample` component, which demonstrates how several components can be pulled together on a single page. This is currently in a proof-of-concept phase and will be expanded, along with additional variants (news article, event page, landing page and so on).

The current template includes:

- **PageHeader**: Complete UNDRR branding with logo, language switcher, and colored decoration stripe
- **MegaMenu**: Navigation menu with multiple sections and banner content
- **Hero**: Primary hero section with customizable content
- **Tab**: Tabbed content sections (both default and stacked variants)
- **VerticalCard**: Grid of content cards
- **Footer**: Complete footer component

### Analytics and messaging

Include these scripts on all UNDRR pages:

```html
<!-- UNDRR Analytics - GA4 bootstrap and enhancements -->
<script
  src="https://assets.undrr.org/static/analytics/v1.0.0/google_analytics_enhancements.js"
  defer
></script>

<!-- UNDRR Critical Messaging - emergency broadcasts -->
<script src="https://messaging.undrr.org/src/undrr-messaging.js" defer></script>
```
