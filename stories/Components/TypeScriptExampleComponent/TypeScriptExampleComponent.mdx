## Working with non-TypeScript code

When integrating with JavaScript components:

```typescript
// Type declarations for existing JS components
declare module 'javascript-component' {
  export interface Props {
    value: string;
    onChange: (value: string) => void;
  }
  export default function Component(props: Props): JSX.Element;
}
