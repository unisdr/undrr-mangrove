## What this component demonstrates

- **Typed props** with a union type for `type` (`'Primary' | 'Secondary'`) instead of a bare `string`
- **Optional props** with `onClick?: () => void`
- **JSDoc comments** on the component and its props (visible in Storybook's controls panel)
- **Explicit return type** (`JSX.Element`) on the component function
- **CSF3 stories** using `Meta` and `StoryObj` from `@storybook/react`

## Working with non-TypeScript code

When importing existing JavaScript components from TypeScript files, TypeScript infers types from PropTypes and JSDoc where possible. For components without PropTypes, you can add a local type declaration:

```typescript
// types/SomeJsComponent.d.ts
declare module '../path/to/SomeJsComponent' {
  export interface Props {
    value: string;
    onChange: (value: string) => void;
  }
  export default function SomeJsComponent(props: Props): JSX.Element;
}
```
