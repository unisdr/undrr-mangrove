# Chips

Small interactive tags used to show active filters, selections, or categories. Chips can include a dismiss icon to let users remove them.

## When to use

- Showing which filters or search terms are currently active
- Letting users remove individual selections without clearing everything
- Labelling active categories or tags in a content listing

## States

**Default (with dismiss icon)** — The chip shows a label and a close icon. Clicking the icon removes the filter or selection.

**Hover (with dismiss icon)** — The chip changes visual appearance when the user hovers over it, signalling it is interactive.

**Default (no dismiss icon)** — The chip shows a label only. It communicates a current selection or category but does not allow removal.

**Hover (no dismiss icon)** — Same visual change as the hover state above; no dismiss action is available.
