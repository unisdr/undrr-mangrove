# Mangrove hero (sub)

The Mangrove Hero components are page sections with information image and overlay colors (primary, secondary, tertiary and quaternary).

### Overview

The Mangrove Hero component defines a section of information with some items. It is a multi-usage component that creates sections that are usually highlighting some kind of content.

#### When to use:

When a page section with data, CTA links and image is to be displayed.

### Formatting

#### Default

It consists of a title, paragraph, CTA links (primary, secondary and tertiary),labels, detail links, image and overlay colors (primary, secondary, tertiary and quaternary).

### Content

There are two types of Mangrove Hero components: Mangrove Hero Parent and Mangrove Hero Child.

### Usage

- Grab the HTML from the HTML tab.

### Interactions

- By clicking on the title and/or CTA Link, the associated page is opened.
