# Mangrove hero (sub)

The Mangrove Hero components are page sections with information image and overlay colors (primary, secondary, tertiary and quaternary).

### Overview

The Mangrove Hero component defines a section of information with some items. It is a multi-usage component that creates sections that are usually highlighting some kind of content.

#### When to use:

When a page section with data, call-to-action buttons, and an image is to be displayed.

### Formatting

#### Default

It consists of a title, paragraph, call-to-action buttons (primary, secondary, and tertiary), labels, detail links, image, and overlay colors (primary, secondary, tertiary, and quaternary).

### Content

There are two types of Mangrove Hero components: Mangrove Hero Parent and Mangrove Hero Child.

<Canvas of={ChildHeroStories.HeroChild} meta={ChildHeroStories} />

### Usage

- Grab the HTML from the HTML tab.

### Interactions

- By clicking on the title or a call-to-action button, the associated page is opened.
