# Fetcher

The `Fetcher` component is a flexible data-fetching utility for React applications.

## Props

- **`api`**: `string` - The URL of the API endpoint. _(required)_
- **`render`**: `function` - A function to render the UI based on the fetched data. _(required)_
- **`queryParams`**: `object` - Additional query parameters to send with the API request. _(optional)_
- **`loadingLabel`**: `string` - A label to display while loading. _(optional)_

## Stories

### Default

This is the default usage of the `Fetcher` component.

<Canvas of={FetcherStories.Default} meta={FetcherStories} />

<Source
  language="jsx"
  code={`
import Fetcher from './Fetcher';

const Default = () => {
const renderData = ({ isLoading, data }) => (

<div>
  {isLoading && <p>Loading data...</p>}
  <pre>{JSON.stringify(data, null, 4)}</pre>
</div>
);

return (

<Fetcher
  api="https://sfvc-migration.undrr.org/api/v2/content/commitments-metrics"
  queryParams={{ items_per_page: 1000 }}
  loadingLabel="Loading data"
  render={renderData}
/>
); };

export default Default;
`}
/>
