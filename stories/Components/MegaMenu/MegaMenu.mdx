# Mega menu

The Mega Menu component provides a navigation menu with multiple levels and sections.

> **Note:** This component is currently in beta and subject to breaking changes in future releases.

## With logo (branded variant)

For sub-branded sites (e.g. PreventionWeb), pass `logoSrc` to display an inline logo at the start of the nav strip. This eliminates the need for a separate logo row between the PageHeader and navigation, creating a compact two-tier header.

Use this variant when the site has its own brand identity but still sits under the UNDRR umbrella (the dark PageHeader bar above carries the parent identity).

## Usage

```jsx
import { MegaMenu } from '@components';

// Default (no logo)
<MegaMenu sections={menuItems} />

// Branded variant (with logo)
<MegaMenu
  sections={menuItems}
  logoSrc="https://assets.undrr.org/static/logos/pw/pw-logo.svg"
  logoAlt="PreventionWeb"
  logoHref="https://www.preventionweb.net"
/>
```

## Props

| Prop       | Type   | Default | Description                                                           |
| ---------- | ------ | ------- | --------------------------------------------------------------------- |
| sections   | array  | []      | Array of menu sections to display                                     |
| delay      | number | 300     | Delay in milliseconds before closing menu on mouse leave              |
| hoverDelay | number | 180     | Delay in milliseconds before opening menu (prevents accidental opens) |
| logoSrc    | string | —       | Image URL for the logo. When provided, enables the branded variant    |
| logoAlt    | string | ''      | Alt text for the logo image                                           |
| logoHref   | string | '/'     | Link target for the logo (typically the sub-brand homepage)           |
| logoWidth  | number | —       | Optional explicit width for CLS prevention                            |
| logoHeight | number | —       | Optional explicit height for CLS prevention                           |
