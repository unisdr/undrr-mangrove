# Mega menu

The Mega Menu component provides a navigation menu with multiple levels and sections.

> **Note:** This component is currently in beta and subject to breaking changes in future releases.

## With logo (branded variant)

Sub-branded sites (PreventionWeb, MCR2030, Global Platform, etc.) can pass `logoSrc` to put a logo inline with the nav items. This replaces the separate logo row you'd otherwise need between the PageHeader and navigation, so the header stays two tiers instead of three.

## Usage

```jsx
import { MegaMenu } from '@components';

// Default (no logo)
<MegaMenu sections={menuItems} />

// Branded variant (with logo)
<MegaMenu
  sections={menuItems}
  logoSrc="https://assets.undrr.org/static/logos/pw/pw-logo.svg"
  logoAlt="PreventionWeb"
/>
```

### Multilingual sites and language-prefixed logo links

`logoHref` defaults to `'/'`. The component doesn't know or care about language prefixes (`/ar/`, `/fr/`, etc.) because URL structure varies between CMSs.

In Drupal, have the Twig template output the language-prefixed root into a data attribute. The wrapper script reads it and passes it as `logoHref`. Example for an Arabic page:

```html
<div data-mg-mega-menu
     data-mg-mega-menu-logo="https://assets.undrr.org/static/logos/pw/pw-logo.svg"
     data-mg-mega-menu-logo-alt="PreventionWeb"
     data-mg-mega-menu-logo-href="/ar/">
</div>
```

Twig is the right place to resolve the prefix. The React component just renders whatever href it receives.

## Props

| Prop       | Type   | Default | Description                                                                                              |
| ---------- | ------ | ------- | -------------------------------------------------------------------------------------------------------- |
| sections   | array  | []      | Array of menu sections to display                                                                        |
| delay      | number | 300     | Delay in milliseconds before closing menu on mouse leave                                                 |
| hoverDelay | number | 180     | Delay in milliseconds before opening menu (prevents accidental opens)                                    |
| logoSrc    | string | —       | Image URL for the logo. When provided, enables the branded variant                                       |
| logoAlt    | string | ''      | Alt text for the logo image                                                                              |
| logoHref   | string | '/'     | Where the logo links to. For multilingual sites, pass the language-prefixed root from the CMS (e.g. `/ar/`) |
| logoWidth  | number | —       | Optional explicit width for CLS prevention                                                               |
| logoHeight | number | —       | Optional explicit height for CLS prevention                                                              |
