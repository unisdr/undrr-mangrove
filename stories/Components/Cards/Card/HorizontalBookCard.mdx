# Mangrove card (horizontal book)

The Mangrove Horizontal Book Card components are cards with information and/or image. This is ideally suited for publications. The component leverage the Horizontal Card CSS.

### Overview

The Mangrove Horizontal Book Card component defines a card of information with some items. It is a multi-usage component that creates boxes that are usually teasing some kind of content.

#### When to use:

When a card with data, CTA links and image is to be displayed.

### Formatting

#### Default

It consists of a title, paragraph, CTA links (primary, secondary and tertiary), labels, social share link and/or image.

### Content

<Canvas of={HorizontalBookCardStories.DefaultHorizontalBookCard} meta={HorizontalBookCardStories} />

### Usage

- Grab the HTML from the HTML tab.

### Interactions

- Hover state applies to the image only.
- By clicking on the CTA Link, the associated page is opened.
