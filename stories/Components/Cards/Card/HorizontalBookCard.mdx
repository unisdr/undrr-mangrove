# Horizontal book card

A horizontal card variant for publications. Displays a portrait cover image on the left with title, summary, labels, and a CTA on the right. Ideal for detailed publication listings where you need both a cover image and descriptive text.

## When to use

- Publication detail listings with summaries
- Resource pages where cover art is important alongside descriptive text
- Search results for documents, reports, and brochures

## Anatomy

- **Cover image** (optional): Portrait-oriented, constrained to 200px on mobile, natural width on desktop
- **Labels** (optional): Content type or category tags
- **Title**: Linked heading with blue color, underline, and chevron indicator
- **Summary**: Body text, supports inline HTML links
- **CTA button** (optional): Primary action button

The entire card is clickable via a stretched link on the title.

## Color variants

Use the `variant` prop to change the title accent color.

| Variant | Class | Color |
|---------|-------|-------|
| `primary` (default) | — | `$mg-color-interactive` (blue) |
| `secondary` | `mg-card--secondary` | `$mg-color-secondary` (orange) |
| `tertiary` | `mg-card--tertiary` | `$mg-color-tertiary` (black) |
| `quaternary` | `mg-card--quaternary` | `$mg-color-quaternary` (red) |

## Title link visibility

Card titles display blue color, underline, and a chevron indicator by default. Add `mg-card--plain-title` to disable these signals.

**In React:**

```jsx
<HorizontalBookCard data={data} className="mg-card--plain-title" />
```

**In Drupal templates or static HTML:**

```html
<article class="mg-card mg-card__hc mg-card-book__hc mg-card--plain-title">
  ...
</article>
```

## Responsive behavior

| Breakpoint | Layout |
|------------|--------|
| Mobile (< 640px) | Stacked: cover image (max 200px wide) above content |
| Tablet (640px+) | Side-by-side: 160px cover image, content fills remaining width |

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `data` | Array | Required | Array of card data objects |
| `variant` | String | `'primary'` | Color variant: `'primary'`, `'secondary'`, `'tertiary'`, `'quaternary'` |
| `className` | String | — | Additional CSS classes on the card element |

### Data object properties

| Property | Type | Description |
|----------|------|-------------|
| `title` | String | Card heading text |
| `link` | String | URL for the title link and stretched link |
| `imgback` | String | Cover image URL |
| `imgalt` | String | Alt text for the cover image |
| `summaryText` | String | Body text (HTML supported) |
| `label1` | String | First category label |
| `label2` | String | Second category label |
| `button` | String | CTA button label |
