# Book card

A compact card variant for publications and brochures. Displays a portrait cover image (3:4 aspect ratio) with a title below. Designed to fit 5-6 across in a row on desktop.

## When to use

- Publication and report listings
- Library/resource pages with cover images
- Any grid of items with portrait-oriented visuals (book covers, posters, brochures)

## Anatomy

- **Cover image**: 3:4 aspect ratio with a subtle border
- **Title**: Linked heading with smaller font size than other card variants

The card has no summary text or labels. The entire card is clickable via a stretched link on the title.

## Color variants

Use the `variant` prop to change the title accent color.

| Variant | Class | Color |
|---------|-------|-------|
| `primary` (default) | — | `$mg-color-interactive` (blue) |
| `secondary` | `mg-card--secondary` | `$mg-color-secondary` (orange) |
| `tertiary` | `mg-card--tertiary` | `$mg-color-tertiary` (black) |
| `quaternary` | `mg-card--quaternary` | `$mg-color-quaternary` (red) |

## Title link visibility

Card titles display blue color, underline, and a chevron indicator by default. Add `mg-card--plain-title` to disable these signals.

**In React:**

```jsx
<BookCard data={data} className="mg-card--plain-title" />
```

**In Drupal templates or static HTML:**

```html
<article class="mg-card mg-card__vc mg-card__book mg-card--plain-title">
  ...
</article>
```

## Sizing

The book card is constrained to `200px` max-width by default, expanding to `400px` on desktop. When used in a grid, let the grid column width control the card size.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `data` | Array | Required | Array of card data objects |
| `variant` | String | `'primary'` | Color variant: `'primary'`, `'secondary'`, `'tertiary'`, `'quaternary'` |
| `className` | String | — | Additional CSS classes on the card element |

### Data object properties

| Property | Type | Description |
|----------|------|-------------|
| `title` | String | Card heading text |
| `link` | String | URL for the title link and stretched link |
| `imgback` | String | Cover image URL |
| `imgalt` | String | Alt text for the cover image |
