# Mangrove card (book)

🚨 Warning this is a non-canonical conceptual component. 1 March 2023. 🚨

### Overview

This is particularly suited for publications and brochures.

It is designed to have more on a row, perhaps 5 or 6 on desktop.

### Formatting

The Book Card variant is more simplistic in content, with a smaller font size and no summary text or labels.

It's images have a 3x4 in aspect ratio.

<Canvas of={BookCardStories.DefaultBookCard} meta={BookCardStories} />

### Usage

- Grab the HTML from the HTML tab.

### Interactions

- Hover state applies to the image only.
- By clicking on the CTA Link, the associated page is opened.
