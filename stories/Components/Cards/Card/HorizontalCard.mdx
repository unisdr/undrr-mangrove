# Horizontal card

A content teaser card in landscape/side-by-side layout. Image on the left, text on the right. Well suited for content-heavy listings where the image is secondary.

## When to use

- List-style content pages (search results, filtered listings)
- Layouts where vertical space is limited and horizontal reading flow is preferred
- Content with longer titles or summaries that benefit from a wider text area

## Anatomy

- **Image**: Displayed at left on desktop (160px or 330px wide), stacked on top on mobile (300px tall)
- **Labels**: Content type or category tags
- **Title**: Linked heading with blue color, underline, and chevron indicator
- **Summary**: Body text, supports inline HTML links
- **CTA button**: Primary action button

The entire card is clickable via a stretched link on the title.

## Color variants

Use the `variant` prop to change the title accent color. These match the Hero component color system and adapt per theme.

| Variant | Class | Color |
|---------|-------|-------|
| `primary` (default) | — | `$mg-color-interactive` (blue) |
| `secondary` | `mg-card--secondary` | `$mg-color-secondary` (orange) |
| `tertiary` | `mg-card--tertiary` | `$mg-color-tertiary` (black) |
| `quaternary` | `mg-card--quaternary` | `$mg-color-quaternary` (red) |

Use the variant control below the canvas to preview each option.

## Title link visibility

By default, card titles display three signals that they are clickable:

1. **Blue color** matching `$mg-color-interactive` (adapts per theme)
2. **Underline** with `3px` offset
3. **Chevron indicator** (CSS border triangle via `::after`)

On hover, the underline thickens to `2px`. In RTL layouts, the chevron flips direction automatically.

### Opting out

Add `mg-card--plain-title` to disable all link signals while keeping the card clickable.

**In React:**

```jsx
<HorizontalCard data={data} className="mg-card--plain-title" />
```

**In Drupal templates or static HTML:**

```html
<article class="mg-card mg-card__hc mg-card--plain-title">
  ...
</article>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `data` | Array | Required | Array of card data objects |
| `variant` | String | `'primary'` | Color variant: `'primary'`, `'secondary'`, `'tertiary'`, `'quaternary'` |
| `className` | String | — | Additional CSS classes on the card element |

### Data object properties

| Property | Type | Description |
|----------|------|-------------|
| `title` | String | Card heading text |
| `link` | String | URL for the title link and stretched link |
| `imgback` | String | Image URL |
| `imgalt` | String | Alt text for the image |
| `summaryText` | String | Body text (HTML supported) |
| `label1` | String | First category label |
| `label2` | String | Second category label |
| `button` | String | CTA button label |

## Responsive behavior

| Breakpoint | Layout |
|------------|--------|
| Mobile (< 640px) | Stacked: image on top (300px tall), content below |
| Tablet (640px+) | Side-by-side: 160px image, content fills remaining width |
| Desktop (1024px+) | Side-by-side: 330px image, content fills remaining width |
