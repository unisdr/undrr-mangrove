# Mangrove card (vertical)

The Mangrove Vertical Card components are cards with information and/or image.

### Overview

The Mangrove Vertical Card component defines a card of information with some items. It is a multi-usage component that creates boxes that are usually teasing some kind of content.

#### When to use:

When a card with data, CTA links and image is to be displayed.

### Formatting

#### Default

It consists of a title, paragraph, CTA links (primary, secondary and tertiary), labels, social share link and/or image.

It's image in 16x9 in aspect ratio.

### Content

There are two types of Mangrove Vertical Card components: Mangrove Vertical Card with Image and Mangrove Vertical Card without Image.
