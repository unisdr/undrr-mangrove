# Stats cards

(NOTE: This is yet to be adapated to the UNDRR system and styles.)

A stats card contains the value, label, and a one-line description of a statistic

### Overview

A stats card highlights an individual statistic. It contains a value, label, and a short description.

Stats cards are usually displayed as a part of a collection of similar cards highlighting various statistics

A card is intended to be modular. This means that you can vary the order of cards in a collection without destroying any individual card's meaning.

### When to use:

To display and highlight multiple statistics about a subject/country

### Formatting

#### Default

The stats card consists of a value, label, and short description. An accent color overlay is displayed on hover.

### Content

The stats card requires at a minimum a value and label. A short description can also be added optionally.
Extra-large and large cards can also have a title

### Behaviors

#### States

The stats card has defaulted and hovers states.

There are 4 size variations of the card: Small, medium, large and extra-large

The following size modifiers are available:

- Extra large
- Large
- Medium
- Small

### Usage

- Choose a different variation from the control tab on canvas.
- Copy the HTML from the HTML tab in the canvas
- to dynamically adjust Stat text size to its container init script with `fitText(selector, options` where `options` is an associative array containing maximum sizes for th stat figure for desktop and mobile, i.e. `{desktop: 110, mobile: 80}` for standard medium card}

### Interactions

The hover state is applied to the entire card
