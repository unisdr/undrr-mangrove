# Textarea

A multi-line text field for longer user input.

## Overview

The textarea component provides a resizable multi-line input area with label, help text, and error state support. It shares the same accessibility patterns as the text input component.

## When to use

- When the user needs to enter multi-line text such as comments, descriptions, or feedback.

## States

- **Default**: Ready for input.
- **Focus**: Blue outline on keyboard focus.
- **Disabled**: Greyed out, non-interactive.
- **Error**: Red border with an error message below.

## Accessibility

- The label is programmatically associated with the textarea via `htmlFor`/`id`.
- Help text and error text are linked via `aria-describedby`.
- Error state sets `aria-invalid="true"` and wraps the error message in `role="alert"`.
- The textarea is resizable vertically by default.
