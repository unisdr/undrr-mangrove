# Checkbox

A checkbox input for selecting one or more options.

## Overview

The checkbox component lets users select or deselect individual options. Each checkbox operates independently and can be grouped inside a `<fieldset>`.

## When to use

- When the user can select zero, one, or multiple options from a set.
- For binary choices like accepting terms or toggling a preference.

## States

- **Default**: Unchecked, red border.
- **Checked**: Red fill with checkmark.
- **Hover**: Purple border on hover.
- **Focus**: Blue outline on keyboard focus.
- **Disabled**: Grey border, non-interactive.

## Accessibility

- The label is programmatically associated with the checkbox via `htmlFor`/`id`.
- For groups of checkboxes, wrap them in a `<FormGroup>` (renders `<fieldset>` with `<legend>`).
- Keyboard users can toggle the checkbox with Space.
