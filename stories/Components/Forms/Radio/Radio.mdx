# Radio button

A radio button for selecting a single option from a set.

## Overview

The radio component allows users to select exactly one option from a group. Radio buttons with the same `name` attribute are mutually exclusive. Wrap groups in a `<FormGroup>` for accessibility.

## When to use

- When the user must select exactly one option from a set.
- For lists of 2-4 options (for more options, consider a select dropdown).

## States

- **Default**: Unselected, red circular border.
- **Selected**: Red fill with white inset.
- **Hover**: Purple border on hover.
- **Focus**: Blue outline on keyboard focus.
- **Disabled**: Grey border, non-interactive.

## Accessibility

- The label is programmatically associated with the radio button via `htmlFor`/`id`.
- For groups, wrap radios in a `<FormGroup>` (renders `<fieldset>` with `<legend>`).
- Keyboard users can navigate between radio buttons with arrow keys.
