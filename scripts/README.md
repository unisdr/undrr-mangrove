# Scripts

Utility scripts for the Mangrove project.

| Directory/File | Purpose |
|----------------|---------|
| `ai-manifest/` | AI-friendly component manifest pipeline. Generates llms.txt, component JSON, and CSS utility reference for AI agents. See [`ai-manifest/README.md`](ai-manifest/README.md). |
| `update-cdn-version.js` | Updates CDN links in documentation files to the current version from `package.json`. Run after each release. |
